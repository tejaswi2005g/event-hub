# Event Hub

A full-stack Event Management System built using the MERN stack.

## Features

- JWT Authentication
- Role-Based Access Control
- Event Registration
- Organizer Dashboard
- Admin Dashboard
- Event Analytics
- QR Code Ticketing
- MongoDB Database

## Tech Stack

### Frontend
- React
- TypeScript
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT
- bcrypt
